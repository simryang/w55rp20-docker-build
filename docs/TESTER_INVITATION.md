## 예상 피드백 및 대응

### "Docker Desktop 설치가 너무 어렵다"
**대응:**
- WINDOWS_TESTING_GUIDE.md에 스크린샷 추가
- 또는 비디오 가이드 제작

### "빌드가 너무 오래 걸린다"
**대응:**
- 정상입니다 (최초 20분)
- 문서에 명시되어 있음
- "이후 빌드는 12초로 빠릅니다" 강조

### "Windows 컨테이너가 안된다"
**대응:**
- 로그 전체 수집
- Windows 환경 직접 테스트 필요
- 또는 "experimental" 라벨 추가

### "Git Bash에서 경로 오류"
**대응:**
- MSYS_NO_PATHCONV=1 확인
- build-windows.sh 디버깅
- Git Bash 버전 확인
